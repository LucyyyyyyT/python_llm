# Agent Instructions

## Project Overview
This is a Python LLM chat agent that lets users query files using natural language.

## Code Style
- Use 4 spaces for indentation
- Add docstrings to all functions
- All functions must have doctests

## Testing
- Run tests with: `coverage run -m pytest --doctest-modules python_llm/`
- Maintain 90%+ coverage at all times

## Git
- Always write descriptive commit messages
- Prefix commits with [docchat]

## Rules
- Never modify files outside the project directory
- Always run doctests after writing a Python file
